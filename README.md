# adnabu Assignment

Based on Given URL download the URL page data as HTML and zip it together and send to email which provided in post request

## Getting Started

To run code in local system clone this repo as

```
	git clone https://github.com/jagamts1/adnabu-assignment.git
```


### Installing

To run server locally 

```
docker-compose up
```

**Notes** 

If you face port issue.Stop the rabbit-mq server before running docker-compose up

```
systemctl stop rabbitmq-server.service
```
